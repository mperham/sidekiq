Simple, efficient message processing for Ruby.

Sidekiq is a drop-in replacement for Resque, using the exact same message format. Because it processes many messages concurrently in each process, it's far more memory efficient. What if 1 process could do the work of 20?

Get it on GitHub Learn more »

Install It

Put sidekiq in your Gemfile: 

gem 'sidekiq'

Install the gem with Bundler: 

bundle install

Call It

Place your worker classes in app/workers: 

# app/workers/hard_worker.rb
class HardWorker
  include Sidekiq::Worker

  def perform(name, count)
    puts 'Doing hard work'
  end
end

In your controller action or model, call HardWorker.perform_async: 

HardWorker.perform_async('bob', 5)

Start It
